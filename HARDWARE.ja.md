# ハードウェアアーキテクチャ資料：水槽管理システム

## 📋 概要

本システムは**多層防御**トポロジーを採用し、入力側（AC）でのノイズフィルタリング、メインバス（DC）での安定化、エッジ側（アクチュエーター）での誘導性スパイク抑制を行います。

```
┌─────────────────────┐     ┌─────────────────────┐     ┌─────────────────────┐
│ レイヤー1: AC        │ ──► │ レイヤー2: DC        │ ──► │ レイヤー3: エッジ    │
│ 入力/保護           │     │ バス/ロジック       │     │ 周辺機器/抑制       │
│                     │     │                     │     │                     │
│ • ACヒューズ        │     │ • 12.53V電源        │     │ • フライバックD     │
│ • NTC（突入電流）   │     │ • LM2596 → 5.1V    │     │ • デカップリングC   │
│ • EMIフィルタ（Y）  │     │ • コンデンサバンク  │     │ • センサー（RJ45）  │
│ • キャニスターリレー│     │ • 8ch MOSFET        │     │ • ポンプ + バルブ   │
└─────────────────────┘     └─────────────────────┘     └─────────────────────┘
```

---

## ⚡ レイヤー1：AC入力・保護（インフラストラクチャ）

電力線ノイズのフィルタリングおよび起動時の突入電流サージからの電源保護を担当します。

### コンポーネント

| コンポーネント | 機能 |
|---|---|
| **ACヒューズ**（3A〜5A） | 短絡保護 |
| **NTC 5D-11** | 突入電流制限器（相に直列） |
| **NDF 222M**（Yコンデンサ） | EMIフィルタ（相–接地、中性–接地） |
| **SSRリレーモジュール 1CH** | キャニスターフィルター独立制御（AC） |

### 配線図

```
[ IEC C14 入力 ]
      │
      ├─── [ 接地ピン ] ────────────────────────┬───► [ 電源 G端子 ]
      │         │                               │
      │         ├───[ NDF 222M ]─── (Lへ)       │
      │         └───[ NDF 222M ]─── (Nへ)       │
      │                                         │
      ├─── [ 中性ピン ] ────────────────────────┼───► [ 電源 N端子 ]
      │                                         └───► [ リレー COMMON ]
      │
      └─── [ 相ピン ] ─── [ ACヒューズ ] ──────┐
                                               │
              ┌────────────────────────────────┘
              │                                │
        [ NTC 5D-11 ]                   [ リレー入力 ]
        （直列）                        （相/NOピン）
              │                                │
        [ 電源 L端子 ]                  [ キャニスター出力 ]
```

## 🔋 レイヤー2：DCバス・ロジック（分配）

電力をロジックレベルに変換し、重負荷スイッチング時のESP32安定性を維持します。

### コンポーネント

| コンポーネント | 機能 |
|---|---|
| **スイッチング電源 180W** | 12.53Vに調整 |
| **ヒューズ T5AL250V** | 8台のポンプとセンサーの物理的保護 |
| **LM2596** | 5.1Vに調整されたステップダウン（ESP32電源） |
| **2× 470µF 16V** | MOSFETの12V入力に並列 |
| **4× 1000µF 10V** | 5V出力に並列（ESP32近傍） |

### 配線図

```
[ 電源 12.53V ]
   │              │
 (V+)           (V─) ─────────────────────────┐ (スターGND)
   │              │                            │
[T5Aヒューズ]     │                            │
   │              │                            │
   ├──────────────┼────────────┐               │
   │              │            │               │
[LM2596 IN+]  [LM2596 IN─]  [MOSFET VIN]  [MOSFET GND]
   │              │            │               │
(出力 5.1V)       │      (2× 470µF 16V)       │
   │              │            │               │
[ESP32 VIN]   [ESP32 GND]     │               │
   │              │      [ 8ch MOSFET ]       │
(4× 1000µF 10V)  │            │               │
   │              └────────────┴───────────────┘
```

> [!IMPORTANT]
> **スターGND**：全ての負側基準は電源のV−端子に直接戻し、グランドループを防止してください。

## 🌊 レイヤー3：周辺機器・抑制（エッジ）

誘導性ノイズ（フライバック）の軽減および長距離ケーブル（1.2m）でのセンサー読み取り安定化を担当します。

### コンポーネント

| コンポーネント | 用途 | 数量 |
|---|---|---|
| **FR154ダイオード**（高速リカバリ） | 給排水ポンプ | 2 |
| **FR154ダイオード**（高速リカバリ） | ペリスタルティックポンプ（肥料4 + プライム1） | 5 |
| **1N5822ダイオード**（ショットキー） | ソレノイドバルブ（チャンネル8） | 1 |
| **10µF / 22µF 50Vコンデンサ** | ケーブル端でのセンサーデカップリング（RJ45） | — |

> [!NOTE]
> **合計：8個のフライバックダイオード** — モーターチャンネルごとに1個（7× FR154 + 1× 1N5822）。すべてケーブル端のモーター近傍に設置。

### 配線図 — アクチュエーター

```
[ MOSFETチャンネル ] ──────── (1.2mワイヤ) ──────┬────────── (ポンプ +)
                                                 │
                                           [ フライバックD ]
                                           (ストライプは+側)
                                                 │
[ バスGND ] ─────────────── (1.2mワイヤ) ────────┴────────── (ポンプ ─)
```

> [!CAUTION]
> **フライバックダイオード**：必ず**ケーブル端**（モーター近傍）に設置してください。1.2mケーブルがアンテナのようにノイズを放射するのを防ぎます。

### 配線図 — 超音波センサー（JSN-SR04T）

センサーは5Vで動作しますが、ESP32のGPIOは最大**3.3V**です。**ECHO**ピンにはGPIO19に入力する前に5V → 3.3Vに降下させる分圧回路が必要です。

| コンポーネント | 機能 |
|---|---|
| **1kΩ抵抗**（R1） | ECHOとGPIO19の間に直列 |
| **2kΩ抵抗**（R2） | GPIO19とGND間のプルダウン |

**計算式：** `Vout = 5V × R2 / (R1 + R2) = 5 × 2k / 3k = 3.33V` ✅

```
                           ┌──────────────────┐
  ESP32 GPIO18 (TRIG) ─────┤ TRIG    JSN-SR04T│
                           │                  │
  ESP32 5V ────────────────┤ VCC              │
                           │                  │
  GND ─────────────────────┤ GND              │
                           │                  │
                           │ ECHO ────┐       │
                           └──────────┼───────┘
                                      │
                                 [ R1 = 1kΩ ]
                                      │
                    ESP32 GPIO19 ──────┤
                                      │
                                 [ R2 = 2kΩ ]
                                      │
                                    (GND)
```

> [!WARNING]
> **ECHOを分圧回路なしでESP32に直接接続しないでください。**5VのECHO信号はESP32のGPIOを永久に損傷する可能性があります。

### 配線図 — 静電容量式水位センサー（XKC-Y25-NPN）

ガラス/チューブ壁越しに液体の存在を検知する非接触センサー。NPN オープンコレクタ出力、ESP32の3.3Vと直接互換性があります。

| センサーピン | 接続先 |
|---|---|
| **VCC**（赤） | ESP32 3.3V |
| **GND**（黒） | GND |
| **OUT**（黄） | GPIO 4（`INPUT_PULLUP`） |

```
  ESP32 3.3V ─────────── VCC（赤）
                                            ┌────────────────┐
  ESP32 GPIO4 ─────────── OUT（黄） ───────┤ XKC-Y25-NPN    │
   (INPUT_PULLUP)                          │ （ガラス外側   │
                                            │  に貼付）      │
  GND ─────────────────── GND（黒） ───────┤                │
                                            └────────────────┘
```

> [!TIP]
> **外部抵抗は不要。**ESP32の内部プルアップ（約45kΩ）でNPNオープンコレクタに十分です。センサーは水槽ガラスの**外側**、希望する最大水位に貼り付けます。

## 🛠️ 安全な実装上の注意点

1. **ハンダ付け優先順位** — NTCは動作中に発熱するため、耐熱熱収縮チューブで絶縁する必要があります。

2. **コンデンサの極性** — すべての電解コンデンサの負極ストライプを確認してください（特に5.1Vで動作する1000µF 10V）。

3. **フライバックダイオード** — 必ずケーブル端（モーター近傍）に設置し、1.2mケーブルがアンテナのようにノイズを放射するのを防いでください。

4. **スターGND** — すべての負側基準は電源のV−端子に直接戻し、グランドループを防止してください。
